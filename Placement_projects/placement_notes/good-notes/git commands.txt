Git commands

git checkout . - Removes Unstaged Tracked files ONLY [Type 2]

git clean -f - Removes Unstaged UnTracked files ONLY [Type 3]

git reset --hard - Removes Staged Tracked and UnStaged Tracked files ONLY[Type 1, Type 2]

git stash -u - Removes all changes [Type 1, Type 2, Type 3]

git branch -vv  Lists all branches on local repo

git remote prune origin -removes branches from the local repo that no longer exist in the remote repo (cleans up branches)

git fetch  -retrives updates from the remote repo
